Useful Plugin to Display MailChimp Newsletter Subscription Widgets in Your Site.

== Description ==
<p>
This WordPress Plugin Allows You to Display MailChimp Newsletter Subscription Widgets in Your Site.

Its a Small but very useful Plugin to Capture Email List Using MailChimp Service. You can place this widget in any location of your site i.e. Sidebar, Footer, Header areas etc...
</p> 

== Installation ==

1. Download Archive & Unpack It.
2. Upload (& overwrite) the /my-mailchimp/ folder & its content to the /plugins/ folder. 
3. Activate the plugin in your Plugin Administration page.
4. Drag the "Weengle : MailChimp Newsletter Widget" on your Sidebar or any Desired Location.
5. Configure your "My Mailchimp Widget".

== Configuration ==
1. Set the Title
2. Set the Sub Title
3. Enter valid Form Action URL (You Can get it from your mailchimp Account while Creating Subscription form).
4. Set Email Field Name.
5. The Last Step is to Save the Settings by clicking the "Save" Button.

== Frequently Asked Questions ==

= Can we Display First Name, Last Name or any other fileds to the Form? =
Its an Initial release of the Plugin. We will keep Updating this Plugin and definitely add More than One Fields in Our Upcoming releases.

= Users are Not Getting MailChimp Subscription Confirmation Messages? =

Sometimes Due to problem with mailing Services, MailChimp's Subscription Confirmation mail won't reach at its Destination. This will be fixed by The MailChimp Team.
